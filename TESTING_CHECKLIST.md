# ✅ Чеклист тестирования

## 🧪 Обязательные тесты:

### 1. Базовый input()
```python
name = input("What's your name? ")
print(f"Hello, {name}!")
```

**Ожидается:**
- [ ] Текст "What's your name?" появляется в консоли
- [ ] Поле ввода появляется внизу
- [ ] После ввода имени - ответ "Hello, [имя]!"

### 2. Игра с циклом
```python
def game():
    print("Locked Room Game")
    print("Commands: look, take, open, quit")
    has_key = False
    
    while True:
        cmd = input("\n> ").strip().lower()
        
        if cmd == "look":
            print("You see a key" if not has_key else "Nothing here")
        elif cmd == "take":
            has_key = True
            print("Took key!")
        elif cmd == "open":
            if has_key:
                print("YOU WIN!")
                break
            print("Door locked!")
        elif cmd == "quit":
            break

game()
```

**Ожидается:**
- [ ] Весь текст игры виден в консоли
- [ ] Поле ввода работает на каждой итерации
- [ ] История команд сохраняется (можно прокрутить вверх)
- [ ] Игра завершается корректно

### 3. Turtle Graphics (проверка что не сломали)
```python
import turtle

t = turtle.Turtle()
t.speed(0.5)

for i in range(4):
    t.forward(100)
    t.right(90)

turtle.done()
```

**Ожидается:**
- [ ] Canvas появляется в консоли
- [ ] Квадрат рисуется с анимацией
- [ ] Черепашка видна и поворачивается

### 4. Множественный input
```python
print("=== Quiz ===")

q1 = input("Q1: 2+2? ")
q2 = input("Q2: Sky color? ")
q3 = input("Q3: Python symbol? ")

print(f"\nAnswers: {q1}, {q2}, {q3}")
```

**Ожидается:**
- [ ] Все 3 вопроса работают
- [ ] Ответы собираются правильно
- [ ] Итоговый вывод корректный

### 5. Error handling
```python
age = input("Your age: ")
print(int(age) + 10)  # Намеренная ошибка если введут не число
```

**Ожидается:**
- [ ] Если ввести "abc" - красивая ошибка
- [ ] Ошибка показывается в панели Errors
- [ ] Консоль остаётся читаемой

## 🎯 Дополнительные проверки:

### UI/UX:
- [ ] Поле ввода имеет золотую границу сверху
- [ ] Промпты выделены золотым
- [ ] Ответы показаны с ▶ стрелкой
- [ ] Автоскролл работает
- [ ] Кнопка Clear очищает консоль и input

### Keyboard:
- [ ] Enter отправляет ввод
- [ ] Backspace работает в поле
- [ ] Ctrl+C/V работают
- [ ] Tab фокусирует поле ввода

### Edge cases:
- [ ] Пустой input (Enter без текста) - не крашит
- [ ] Очень длинный input - не ломает UI
- [ ] Быстрый input (spam Enter) - не ломается
- [ ] Закрытие/перезагрузка - состояние не теряется

## 🐛 Известные ограничения:

1. **Нельзя остановить бесконечный цикл без input**
   - Если код зависнет без input() - нужна перезагрузка страницы
   
2. **Turtle + input в одной программе**
   - Может конфликтовать из-за event loop
   - Рекомендация: используйте отдельно

3. **Очень быстрые input (< 50ms)**
   - Event loop может не успеть
   - Обычно не проблема для человека
